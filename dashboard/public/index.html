About the Project

This project compares time-series encoding methods for stress detection using CNN architectures. We encode physiological signals (ECG, EDA, EMG, etc.) from the WESAD dataset into images using:

  • GAF Gramian Angular Field
  • MTF Markov Transition Field
  • RP Recurrence Plot

Key Features

  • ✨ Channel Attention (CBAM/SE) for interpretability
  • ✨ Leave-One-Subject-Out cross-validation
  • ✨ EfficientNet for efficient inference
  • ✨ Grad-CAM visualization
  • ✨ Comprehensive metrics (AUC, F1, etc.)
