Decoding the language

VitalAura is a deep learning system that reads raw ECG waveforms, encodes them as multi-dimensional images, and classifies cardiac arrhythmias with attention-guided precision. Built on ResNet-18 and CBAM, trained on over 109,000 heartbeats from the MIT-BIH Arrhythmia Database.

A different way to see the heart

Traditional ECG classification treats cardiac signals as one-dimensional sequences. VitalAura takes a fundamentally different approach: it transforms each heartbeat into a rich, multi-layered image that reveals patterns invisible to conventional methods.

Three mathematical transforms capture complementary aspects of the signal. A Gramian Angular Field encodes temporal correlations. A Recurrence Plot maps the dynamical structure. A Markov Transition Field captures probabilistic state changes. Stacked together as colour channels, they form a single composite image that carries far more information than the raw waveform alone.

Architecture

ResNet-18 Backbone

A residual network pre-trained on ImageNet provides the feature extraction foundation. Skip connections allow deep hierarchical learning from the fused encoding images without gradient degradation.

CBAM Attention

The Convolutional Block Attention Module applies channel and spatial attention at each residual block, learning to emphasize the most diagnostically significant morphological patterns.

Five-Class Output

The classifier distinguishes Normal, Supraventricular, Ventricular, Fusion, and Unknown beat types following the AAMI standard for heartbeat categorization.
