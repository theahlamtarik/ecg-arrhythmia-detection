About the Project

Transforming ECG Signals into Actionable Insights

This project presents a complete deep learning pipeline for automatic classification of ECG heartbeats from the MIT-BIH Arrhythmia Database into five clinically relevant categories, using innovative time-series-to-image encoding and attention-enhanced CNNs.

Multi-Modal Encoding

Combines GAF, Recurrence Plots, and Markov Transition Fields into RGB images

CBAM Attention Mechanism

Channel & Spatial attention modules focus on the most relevant cardiac features

Explainable AI (Grad-CAM)

Visual attention maps show exactly where the model looks for each prediction
